## TFFinder
* An application to view computer files on mobile devices; 
* Effectively solve the problem of insufficient storage space of mobile devices.
* 一个在移动设备上查看电脑文档的应用软件；
* 有效解决移动设备存储空间不足的问题。

## Build in
* Mac OS X 10.10
* Xcode 8.2.1
* iOS 8.0
